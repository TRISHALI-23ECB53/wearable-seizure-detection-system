# 🧠 Wearable Seizure Detection System

A low-cost wearable healthcare system for real-time seizure detection using ESP32, MPU6050, and MAX30102 sensors.

## 🔧 Features
- Real-time seizure monitoring
- Motion + heart rate detection
- Instant alert generation
- IoT-based monitoring
- Wearable prototype design

## 🛠 Hardware Used
- ESP32
- MPU6050
- MAX30102
- OLED Display

## 📊 Results
- Detection Accuracy: ~80%
- Alert Time: 8–12 sec
- Real-time monitoring validated

## 🚀 Future Scope
- Mobile app integration
- AI-based prediction
- Cloud dashboard
- Compact wearable design
